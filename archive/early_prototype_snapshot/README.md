# Early Prototype Snapshot

This directory is a curated archive of the authored files that existed in `/Users/iwosmura/Downloads/hack4sages` but were not exact copies of files already present in the cleaned project repo.

What was kept:

- exploratory notebooks from the original top-level prototype
- TauREx and Ariel notebook-era experiments
- notebook-side model code and helper scripts
- small dataset notes and manifests from the old `cross/` bundle
- Garnet experiment notebooks and reports

What was intentionally left out:

- raw datasets and large HDF5 / Parquet assets
- generated outputs, checkpoints, and notebook dump directories
- virtual environments, caches, and checkpoint folders
- exact duplicate files already present elsewhere in the repo

The internal layout mirrors the old snapshot on purpose so it is easy to trace provenance back to the pre-cleanup structure.
