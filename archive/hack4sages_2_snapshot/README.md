# hack4sages 2 Snapshot

This directory is the curated import of `/Users/iwosmura/Downloads/hack4sages 2`.

That source folder was not one coherent repo. It was a mixed dump of alternate websites, model experiments, exported figures, research PDFs, baseline code, video source, weights, datasets, node modules, and generated outputs. So this snapshot keeps the parts that are actually useful to preserve and intentionally drops the rest.

What was kept:

- `baseline/`: baseline notebooks and helper scripts
- `exobiome_model/`: training notebooks, audit reports, and planning docs
- `iwo_web/` and `prompt_updated/`: alternate Next.js presentation prototypes without build output
- `video/`: source for the presentation video composition
- `quantum_advantage/`: notes, notebooks, and helper code for the quantum-advantage investigation
- `quantum_model*`: older training scripts and cross-generator notebooks
- `research/`: markdown notes and reference PDFs
- `plots/` and selected `different_shit/` figures: exported presentation assets worth keeping
- `models/ariel_exobiome/`: architecture diagram assets that were not already present in the cleaned repo
- `data/eda/`: EDA notebook and notes not otherwise represented here

What was left out:

- datasets, raw bundles, and model weights
- generated outputs and checkpoints
- virtual environments, caches, `node_modules`, `.next`, and `__pycache__`
- random screenshots, videos, and other obvious personal scratch files
- exact duplicates of files already present in `/Users/iwosmura/projects/hack4sages`

Use this as provenance and reference material, not as an active working tree.
