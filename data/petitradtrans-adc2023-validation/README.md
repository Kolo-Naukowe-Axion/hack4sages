# pRT ADC2023 Validation Dataset

This directory contains the finalized `20,000`-sample pRT-based ADC2023 validation set.

## Contents

- `ValidationData/`: ADC2023-format spectra, auxiliary table, and ground-truth package.
- `manifest.json`: dataset manifest, checksums, and provenance.
