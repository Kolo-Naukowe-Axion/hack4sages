# ExoBiome Web App

## Project
Quantum biosignature detection web app for HACK-4-SAGES hackathon. Dark space-themed aesthetic.

## Structure
3-page SPA with top navbar:
1. **Landing** - Hero + science explainer + CTA to Planet Explorer
2. **Planet Explorer + Detection** - Timeline → Planet Summary → Submit → 3-column Results → Bridge to Models
3. **Models** - 3 model detail cards + comparison table + about/team footer

## Tech Stack
- Frontend: React/Next.js or static HTML/JS
- Backend: Python (FastAPI/Flask)
- Viz: Chart.js or D3.js for frontend plots
- ML: qiskit, sQUlearn, scikit-learn
